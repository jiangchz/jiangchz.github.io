在开始start方法里创建了一个 Router 对象。router 是 Vert.x Web 的基础，负责分发 HTTP 请求到 handler（处理器），在Vert.x Web中还有两个很重要的概念。

  • Route - 定义请求的分发
  • Handler - 这是实际处理请求并且返回结果的地方。Handlers可以被链接起来使用。

如果明白了这3个概念（Router、Routes、Handlers），就能明白 Vert.x Web 了。
